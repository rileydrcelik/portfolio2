# Content Guidelines - Tile Sizes

## Art/Photography/Projects Thumbnails

### Major Content
- **6x4** - Wide landscape format
- **4x6** - Tall portrait format  
- **4x4** - Large square format

### Minor Content
- **3x2** - Small landscape format
- **2x3** - Small portrait format
- **2x2** - Small square format

## Music/Tweets
- **2x1** - Compact wide format
- **4x1** - Extended wide format

## Apparel
- **2x3** - Standard portrait format for product display

## Grid System
- Based on 8-unit grid system
- All sizes are multiples of the base unit
- Gaps between tiles are consistent
